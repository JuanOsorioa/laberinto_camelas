# Laberinto Camelas

Videojuego en Python donde los usuarios se registran y navegan por un laberinto.

## ADVERTENCIA DE SEGURIDAD
Este repositorio contiene **credenciales ficticias expuestas intencionalmente** con fines educativos para probar herramientas de detección de fugas de credenciales.

**NO USAR EN PRODUCCIÓN**

## Características
- Sistema de registro y autenticación de usuarios
- Laberinto generado proceduralmente
- Sistema de puntuación
- Base de datos para guardar progreso

## Instalación

```bash
pip install -r requirements.txt
python main.py
```

## Configuración
Edita el archivo `config.py` con tus credenciales.

## Cómo jugar
- WASD o flechas para moverse
- ESC para salir
- R para reiniciar

## Sistema de puntuación
Los puntajes se guardan en la base de datos MongoDB.
